# 📊 Predictive Analytics Dashboard

## Overview
This project predicts future sales trends using historical sales data and machine learning.

## Features
- Data Cleaning
- Sales Forecasting
- Interactive Dashboard
- Trend Visualization
- Regression Modeling

## Tech Stack
- Python
- Streamlit
- Scikit-learn
- Plotly
